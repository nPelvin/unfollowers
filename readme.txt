# Unfollowers

Unfollowers is an application that creates a list of people on instagram that don't follow you back.

## Installation & Usage

Steps for installation and usage are as follows.

You will need to have Python 3 installed to run.

1) Download your personal data from instagram in Json format. 
2) Extract the zipped instagram file into add-data-here file.
3) Your unfollowers list will output as unfollowers.txt. Add-data-here file will then be emptied.
